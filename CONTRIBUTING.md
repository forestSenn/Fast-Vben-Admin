# Contributing

Thanks for helping improve Fast Vben Admin.

## Development Flow

1. Create a feature branch from `develop`.
2. Keep changes focused and update related docs.
3. Run the relevant backend, frontend, or Compose checks.
4. Open a pull request with a clear summary and test notes.

## Commit Style

Use Conventional Commits when possible:

```text
feat: add login integration
fix: handle expired token
docs: update deployment guide
chore: update compose config
```
